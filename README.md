# Pulsar.dev

A modern reactive framework for building web applications.

## Packages

This monorepo contains:

- **pulsar** - Core reactive framework
- **@pulsar/transformer** - TypeScript transformer for Pulsar JSX
- **@pulsar/vite-plugin** - Vite plugin for Pulsar
- **@pulsar/design-tokens** - Design system tokens
- **@pulsar/demo** - Demo applications and examples

## Getting Started

```bash
# Install dependencies
pnpm install

# Run demo in development mode
pnpm dev

# Build all packages
pnpm build
```
